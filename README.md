# OneDrive AppIndicator
Gnome3 AppIndicator for [skilion`s OneDrive Free Client](https://github.com/skilion/onedrive).

## Installation
 ```
 git clone git@github.com:fe-lix-werner/onedrive-appindicator.git
 cd onedrive-appindicator
 ./install.sh
